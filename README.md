# CodeAlpha_StudentGradeTracker

## Description
A Java GUI-based application to manage student grades.  
It allows users to input student names and marks, calculate average, highest, and lowest scores, and display a summary report.

## Features
- Java Swing based GUI
- Uses ArrayList to store student data
- Calculates average, highest, and lowest marks
- Displays complete student summary report

## Technology Used
- Java
- Java Swing
